Denis Sopov Studio
==================

![](https://github.com/web4life/DenisSopovStudio/tree/master/design/home.png)


Требования:
-----------

* Apache 2.2.21
* MySQL 5.5.16
* phpMyAdmin 3.4.5
* wordpress v3.2.1
		
Настройки:
----------

* зайти в "localhost/phpmyadmin";
* Создать базу данных "<имя_БД>";
* Импортировать базу из "папка_проекта\app\backup\database.sql";
* Зайти в админку Wordpress
* Wordpress предложит создать файл wp-config.php;
* При создании в автоматическом режиме, необходимо заполнить необходимые поля: база данных = "<имя_БД>", логин = "root", пароль = "", остальные поля заполняются пользователем;
* Заходим в админку Wodrpress;
* Активируем нужную тему "markup";
