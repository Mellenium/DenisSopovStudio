Требования:
	наличие браузера (Google Chrom v14+),
		XAMPP Control Panel v2.5
			с запущенными модулями Apache и MySql,
		Git Bash v1.7.7.1,
		


Скачать всё с репозитория и сохранить в папку ..\xampp\htdocs\wordpress\

Настройки:
	1) В брузере зайти в "localhost/phpmyadmin";
	2) Создать базу данных "wordpress4";
	3) Зайти через браузер в "localhost/wordpress/wp-admin";
	4) Wordpress предложит создать файл wp-config.php;
	5) При создании в автоматическом режиме, необходимо заполнить необходимые поля:
				
			база данных = "wordpress4", логин = "root", пароль = "". 			Остальные поля заполняются пользователем.
	6) Заходим в localhost/wordpress/wp-admin;
	7) Активируем нужную тему "markup";
